⚡ Match Start & Validity
✓
A tennis match is deemed to have started with the first serve of the match.
✓
If a player retires, is disqualified, or a walkover occurs, all markets will be void unless already unconditionally determined.
✓
This includes all undecided point, game, or set-related bets.
🌧 Delays & Suspension
⏸
In case of any delay (rain, darkness…) all markets remain unsettled; trading resumes when the match continues.
⏸
If match is abandoned/suspended before a natural end and does not resume within 5 hours, all bets are Auto Voided.
📋 Scoring & Markets
✓
If penalty points are awarded by the umpire, all bets on that game will stand.
⚠
If markets remain open with an incorrect score significantly impacting prices, Stake reserves the right to void betting.
⚠
If players/teams are displayed incorrectly, Stake reserves the right to void betting.
✓
A match tie-break played as deciding set in Best-of-3 format is counted as the 3rd set. Tie-breaks count as one game for game totals.
🏟 Venue Changes
✗
All bets created prior to a venue change shall be Auto Voided.
📊 Status Summary
Retirement → VOID Walkover → VOID Penalty pts → STAND Delay → SUSPENDED Venue change → VOID
⚡ Match Validity & Abandonment
✓
If a match is postponed, all markets are voided unless it starts on the scheduled day (local time).
✓
If abandoned, all undecided markets are voided unless the match continues within 48 hours of the official start time.
✓
Fully decided markets at time of abandonment (e.g. "First Team to Score") will still be settled.
🌧 Weather & DLS Method
✓
If match is affected by bad weather, bets are settled on the official competition rule including the Duckworth-Lewis (DL) or Jayadevan (VJD) method.
⏸
In case of any delay (rain, etc.) all markets remain unsettled; trading resumes when match continues.
✗
Where no price is quoted for the tie and no winner is determined, all wagers will be voided/cancelled.
👤 Player Props
✗
Bets on any player not in the starting 11 are void.
✓
Player Must Play applies to all pre-match Player Props — the player must participate for bets to stand.
🏟 Venue Changes
✓
If venue changes but the home team retains home designation, bets stand.
✗
If the home and away teams are reversed, all prior bets are void.
⚠ Errors & Integrity
⚠
Stake reserves the right to void bets placed with an incorrect score or due to technical fault/transmission error.
✓
Third-party content (scores, analytics) is provided by external providers. Stake is not responsible for third-party data accuracy.
📊 Status Summary
Abandoned >48h → VOID DLS result → STAND Non-starter prop → VOID Decided markets → SETTLED Delay → SUSPENDED
⚡ Match Validity
✓
If a match is postponed, all markets are voided unless it starts on the scheduled day (local time).
✓
If abandoned and completed within 48 hours of the scheduled start, open bets are settled on the final result.
✗
If not completed within 48 hours, all undecided bets are considered void.
✓
Markets already determined at abandonment (e.g. "First goalscorer") are still settled normally.
🏟 Venue & Opponent Changes
✓
If venue changes but the match is not switched to the opponent's ground and home/away are unchanged, bets stand.
✗
If home and away are reversed, Stake reserves the right to void betting.
✗
If the opponent changes from the listed team, all bets for that match are void.
⏱ Overtime & Markets
✓
3-way Match Betting and Double Chance markets exclude overtime unless stated otherwise.
✓
Markets do not consider overtime unless otherwise stated. Half-time markets are settled on the first half only.
✗
If match is abandoned before half-time, half-time bet is void unless already unconditionally determined.
🎯 Goals & Totals
✓
Total goals count regardless of whether scored before or after the bet was placed (unlike Asian lines).
✗
If game is abandoned, Over/Under bets are void unless outcome is already determined.
✓
Asian Handicap: result before bet placement is ignored for settlement. Abandoned games → bets void.
⚠ Errors & Integrity
⚠
If markets remain open with an incorrect score or match time (>89 seconds off), Stake reserves the right to void betting.
✓
Note: Games with start dates altered well in advance for TV/fixture reasons are not classed as postponed.
📊 Status Summary
Abandoned >48h → VOID Abandoned <48h → SETTLED Opponent changed → VOID Home/Away reversed → VOID Decided mkt → SETTLED
